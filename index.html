Scout Car - Project Arduino

This project was a something I made with a team of 4 for the Thales Project Arduino competition! This was my first experience with a project outside of the classroom so it was extremely fun and I learned alot from it. It is a simple IOT RC car that can be driven using keyboard input via Bluetooth and also has a makeshit "camera" on a pan/tilt servo that moves...

Read more
...corresponding to your gaze data. Gaze data is picked up from a Tobi EyeX camera that uses machine vision to track eye movement. My contribution to the team was the hardware/firmware aspect of the project. I created what is now our final circuit design as well as picked out most of the hardware modules. I also worked with coding the firmware, in this case a Arduino Uno R3.
