Software Projects

Risc-V Processor
This project was assigned to us in my computer organization class. The goal of this project was to implement a fully functional Single Cycle Risc-V Processor then a Pipelined version for our final project that can handle data/control hazards and forwarding. This project was written in SystemVerilog and synthesized using Synopsys.

ZotClicker
This small coding project is a "Cookie Clicker" style UCI inspired game, done for HackUCI 2018. "A hackathon is a place where the only thing limiting what you build is your own creativity - we'll provide the rest. Stay up for 36 hours with the smartest people you'll ever meet and building the coolest things you have created." This project was made more for fun than for the competition as my coding skills are not that developed. I will say that this was an insanely entertaining project because my team and I were able to just create something unique yet full of laughs the whole time.
