Scout Car - Project Arduino

This project was a something I made with a team of 4 for the Thales Project Arduino competition! This was my first experience with a project outside of the classroom so it was extremely fun and I learned a lot from it. It is a simple IOT RC car that can be driven using keyboard input via Bluetooth and also has a makeshift "camera" on a pan/tilt servo that moves...

...corresponding to your gaze data. Gaze data is picked up from a Tobi EyeX camera that uses machine vision to track eye movement. I also worked with coding the firmware, in this case a Arduino Uno R3.

Micromouse - "MouseRat

Two layer micromouse. Built with simple and affordable components. Unique design for modular systems in a smaller, more compact space. This design won 3rd place at the IEEE Southern Area Region 6 Student Design Competition! A Micromouse is a small AI robotic mouse that autonomously solves a 16x16 maze.

Projector Buddy

Proof-of-concept project for a device that can externally attach to projectors and save power using various sensors, internet of things, and machine learning.

Software Projects

Risc-V Processor

System Verilog

This project was assigned to us in my computer organization class. The goal of this project was to implement a fully functional Single Cycle Risc-V Processor then a Pipelined version for our final project that can handle data/control hazards and forwarding. This project was written in SystemVerilog and synthesized using Synopsys.
